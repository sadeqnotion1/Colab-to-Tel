# Skills index

> Registry of project-specific skills the AI can load when a task matches.
> This index **ships empty by design** — author skills as you need them from
> `_template/SKILL.md`, then register them in the table below.

| Skill | When to load it | Path |
|---|---|---|
| _(none yet)_ | — | — |

## How to add a skill
1. Copy `_template/SKILL.md` to `skills/<skill-name>/SKILL.md`.
2. Fill in its trigger description and checklist.
3. Add a row to the table above.

## Rule
If no skill matches the current `NEXT.md` task, say **"none found"** and proceed
mechanically — never invent a skill.
