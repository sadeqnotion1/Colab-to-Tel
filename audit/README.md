# Audit Workspace

This folder contains Day 1 audit artifacts for the world-class repository program.

## Files
- `charter.md`: Audit charter, objectives, scope, success criteria, approvals.
- `scorecard.md`: 0-5 scoring rubric and scoring worksheet.
- `metric-catalog.md`: Metric definitions, formulas, owners, cadence, targets.
- `scope-and-risk-map.md`: System boundaries, dependency map, risk register.
- `governance-cadence.md`: Meeting cadence, decision rights, escalation matrix.
- `day1-decisions-log.md`: Decision log, open questions, action tracker.
- `baseline.md`: Initial measured baseline and prioritized findings.

## Recommended Day 1 Order
1. Fill `charter.md`
2. Finalize `scorecard.md`
3. Confirm data sources in `metric-catalog.md`
4. Complete `scope-and-risk-map.md`
5. Align operating model in `governance-cadence.md`
6. Capture outcomes in `day1-decisions-log.md`

## Rule
No Day 2 baseline work starts until all Day 1 files are at least 90% complete and approved.
